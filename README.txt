Sweet Retreat Website
=====================

Files included:
- index.html     → Main website (open this in a browser to preview)
- styles.css     → Stylesheet placeholder for custom overrides
- script.js      → JavaScript placeholder for custom scripts
- README.txt     → This file

To launch:
1. Open index.html in any web browser to preview locally
2. Upload all files to your web host (GoDaddy, Squarespace, Wix, etc.)
3. Your host will serve index.html automatically as the homepage

To customize:
- Address/phone: Search for "Get Directions" and "Call Us" in index.html
- Social links: Search for "Instagram", "Facebook", "TikTok" in index.html
- Hours: Update the footer section in index.html
- Colors: Edit the :root CSS variables at the top of the <style> block

Need help? Contact your web developer with this folder.
